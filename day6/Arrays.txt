Definition :- An array is a collection of items of same data type stored at contiguous memory locations.
              It is a predefined class.
Syntax :- int a[] = new int[sizeOfAnarray];

Use :- we can store multiple values in a single variable.

Types of arrays :-
1. Single dimensional - it has rows
2. Two dimensional / Multidimensional - it has rows and columns

Object Array - predefined class in java which contains any type of data
Object a[] = new Object[5]
